1. Open the terminal

How to run the algorithm:

When in the part 2 directory in the terminal, type the following:

or

python3 Naive.py spamLabelled.dat spamUnlabelled.dat


The first parameter is the labelled set (training)
The second parameter is the unlabelled set (test)
